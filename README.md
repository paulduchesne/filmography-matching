# Filmography matching

Entity matching between filmographic datasets.

## Method

The process used for matching is based on a suggestion made by Georg Eckes (Bundesarchiv, Berlin) in 2019 during a discussion around practical linked open data strategies for film archives, and possible linking methodologies.

This involves working through creators from the first dataset, identifying a list of attached film titles, generating a list of possible match candidates based on loose name similarity, and identifying a list of film titles for each. For each film title of the original creator, best match scores are collected from each candidate, with a median title score of 100 being judged a successful match.
